# Gamebar - Gaming Marketplace

Gamebar is a full-stack gaming marketplace web application where users can register, login, upload games, manage profile, wishlist games, and admins can manage users and games.

## Tech Stack

### Backend
- Java
- Spring Boot
- Spring Security
- JWT Authentication
- MySQL
- JPA / Hibernate

### Frontend
- HTML
- CSS
- JavaScript

## Features

- User Register/Login
- JWT Authentication
- User Profile
- Add Game
- Edit/Delete Uploaded Games
- Wishlist
- Admin Login
- Admin Dashboard
- Ban/Unban Users
- Delete Users
- Approve Games

## How to Run

### Backend
1. Open backend folder in STS
2. Configure MySQL database in `application.properties`
3. Run Spring Boot application

### Frontend
1. Open frontend folder in VS Code
2. Run using Live Server
3. Make sure backend is running
